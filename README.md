# Lucy
Lucy es una asistente conversacional con inteligencia artificial diseñada para simular una personalidad compleja, emociones, memoria a largo plazo, objetivos propios, aprendizaje activo y un «cuerpo virtual» que siente necesidades (hambre, higiene,). Todo ello alimentado por llamadas a la API de DeepSeek para generar respuestas y analizar contexto.
